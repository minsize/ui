## Group
Компонент используется для группировки контента

```jsx
const Example = () => {
  return (
    <Group>
      <Group.Header>Header</Plug.Header>
      <Group.Container>
        <Cell>Павел Дуров</Cell>
        <Cell>Илон Маск</Cell>
        <Cell>Петя Камушкин</Cell>
      </Group.Container>
      <Group.Footer>
        Footer
      </Group.Footer>
    </Group>
  )
}
```
